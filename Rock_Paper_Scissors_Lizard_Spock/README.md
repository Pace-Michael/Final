Welcome to Rock, Paper, Scissors, Lizard, Spock!

TO PLAY: 
To play the game you must install pygame using the 'pip install' pygame command.

HOW IT WORKS:
The game works by using ramdom number generation to make the computer pick from a list containing Rock, Paper, Scissors, Lizard, Spock.


THE RULES ARE AS FOLLOWS:
    The user will pick one of the following: Rock, Paper, Scissors, Lizard or Spock.
    The computer wil than randomly make a cohoise.

        
HOW TO WIN:
    Rock beats scissors and lizard.
    Paper beats rock and disproves Spock.
    Scissors cuts paper and decapitates lizard.
    Lizard eats paper and poisons Spock.
    Spock vaporizes rock and smashes scissors.
